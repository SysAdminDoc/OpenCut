Welcome to OpenCut Studio

A calmer way to clean dialogue, generate captions, repair shots, and publish edits without leaving Premiere Pro.

In-Panel Workflow Stay inside Premiere with one focused workspace instead of bouncing between tools.
Fast Navigation Jump to recent actions, favorites, and toolsets with the launcher and sidebar.
Results In Context Run the job, review the output, and send finished assets right back into the edit.
1

Select the Footage

Pick a clip from your project bin, use your timeline selection, or drag a file directly onto the panel.

2

Choose the Toolset

Move between Cut, Captions, Audio, Video, Export, Timeline, Search, and Settings from one consistent surface.

3

Run, Review, Auto-Import

Configure the action, let OpenCut process the media, and bring the result back into the current Premiere project.

Quick Tip

Press Ctrl+K any time to open the command palette and jump to recent actions, pinned tools, and the current workspace.
